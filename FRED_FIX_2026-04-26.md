# FRED Integration Fix - 2026-04-26

## Issue Report
**Reported:** User encountered "internal server error" when testing "What is the current Federal Funds Rate?" on the UI (http://136.116.205.168)

## Root Cause Analysis

### Issue 1: Missing FRED_API_KEY in Kubernetes Secret
- **Location:** `k8s/secret.yaml`
- **Problem:** FRED_API_KEY was not defined in the Kubernetes secret
- **Impact:** `settings.fred_api_key` was `None`, causing `create_fred_connector()` in `main.py:46` to return `None`
- **Evidence:** Backend logs would show "FRED_API_KEY not set, FRED connector will be unavailable"

### Issue 2: Tool Dispatch Creating New Connector
- **Location:** `backend/agent/tools.py:545-546`
- **Problem:** The `get_interest_rate` tool was creating a new `FREDConnector()` instance instead of using the `fred_connector` parameter passed from the agent
- **Impact:** Even if Issue 1 was fixed, the tool would still fail because it wasn't using the initialized connector
- **Code:**
  ```python
  # BEFORE (broken):
  elif tool_name == "get_interest_rate":
      from backend.connectors.fred_connector import FREDConnector
      fred = FREDConnector()  # ❌ Creates new instance, ignores parameter
      ...

  # AFTER (fixed):
  elif tool_name == "get_interest_rate":
      if fred_connector is None:  # ✅ Uses passed parameter
          raise AgentError("FRED connector not available. Please configure FRED_API_KEY.")
      ...
      result = await fred_connector.get_current_rate(series_id=series_id, date=date)
  ```

## Fixes Applied

### 1. Added FRED_API_KEY to Kubernetes Secret
**File:** `k8s/secret.yaml`
```yaml
stringData:
  OPENAI_API_KEY: "dummy"
  EXCHANGERATE_API_KEY: "da38a6e0ae77494247ec87cd6abe8823"
  FRED_API_KEY: "49902bd505135a5970742c278a2584a7"  # ✅ Added
```

### 2. Fixed Tool Dispatch to Use Passed Connector
**File:** `backend/agent/tools.py`
- Changed line 545-546 to use `fred_connector` parameter instead of creating new instance
- Added proper error handling when `fred_connector` is `None`

### 3. Updated Deployment Image
